Configuration

Option Description
Select the Worksheet Select the woksheet, in the dashboard, you want to use to extract data.
Extract all worksheet data If selected, it will extract all the data in the worksheet instead of only loading the selected marks.
Endpoint URL URL to the endpoint you want the extension to send data.
Select the Spreadsheet If using the Google Spreadsheet provided, this will set the sheet name where to store de data. The default value is Tableau.
Display Measures If the option is selected, the measures are displayed in the datatable.
Send only selected rows If the option is selected, when uploading data to the endpoint URL, only the selected rows will be sent.
Add Column Add new columns that will be visible in the datatable, to be able to add new information to you data; After writting the column name, click in the Add button. To remove a column, click the Remove button. If you want to change the order the columns are display, click on and drag it up or down to change the order.
Write-back Fields The fields selected will be sent when uploading the data. The other fields will be ignored.
Submit This button saves the configuration.
Clear All This button removes all the columns, sets the Google spreadsheet to the default value and removes the endpoint URL;
